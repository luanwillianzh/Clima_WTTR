# Clima WTTR

Dashboard de previsão do tempo simples e responsivo, desenvolvido com HTML, CSS (Bootstrap 5) e JavaScript puro. Utiliza a API pública [wttr.in](https://wttr.in) para exibir condições climáticas atuais e previsão por hora de qualquer cidade.

## Funcionalidades

- Busca de clima por nome da cidade
- Exibição de temperatura, sensação térmica, umidade e velocidade do vento
- Previsão detalhada para as próximas 24 horas
- Tema claro/escuro (com Bootstrap 5.3)
- Geolocalização aproximada ao carregar a página
- Formulário de contato que envia e-mail com os dados climáticos via Cloudflare Worker

## Tecnologias

- HTML5 + JavaScript (ES2017+)
- [Bootstrap 5.3](https://getbootstrap.com/)
- [wttr.in](https://wttr.in) — API pública de clima
- Cloudflare Workers — envio de e-mails

## Como usar

Basta abrir o arquivo `index.html` em qualquer navegador moderno. Não requer instalação de dependências nem servidor.
